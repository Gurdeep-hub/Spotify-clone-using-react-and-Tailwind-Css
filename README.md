# Spotify Clone

A Spotify-inspired music streaming web application built using React and Tailwind CSS.

## Features
- Responsive UI
- Music player interface
- Playlist display
- Modern Spotify-inspired design

## Tech Stack
- React
- Tailwind CSS
- JavaScript

## Installation

git clone <repo-url>

cd project-folder

npm install

npm run dev
